# Retail Sales Analysis

## Descripción
Este proyecto analiza y predice las ventas de una tienda de retail utilizando técnicas de ciencia de datos.

## Estructura del Repositorio

/retail-sales-analysis
│
├── /data
│ └── dataset.csv # Datos crudos del proyecto
│
├── /notebooks
│ ├── EDA.ipynb # Análisis exploratorio de datos
│ ├── Preprocessing.ipynb # Preprocesamiento de datos
│ └── Benchmarking.ipynb # Modelamiento y evaluación
│
├── /reports
│ ├── classification_report.txt # Reportes de clasificación
│ ├── confusion_matrix.png # Matriz de confusión
│ └── roc_curve.png # Curva ROC
│
├── /presentation
│ └── onepage_presentation.pptx # Presentación resumida del proyecto
│
└── README.md # Documentación del proyecto

## Instrucciones de Instalación

## Instrucciones para Ejecutar

1. Clonar el repositorio:

   git clone https://github.com/tu_usuario/retail-sales-analysis.git
   cd retail-sales-analysis

2. Instalar las dependencias necesarias (se recomienda usar un entorno virtual)

3. Ejecutar los notebooks en orden:

    - notebooks/EDA.ipynb

    - notebooks/Preprocessing.ipynb

    - notebooks/Benchmarking.ipynb

4. Revisar los reportes en la carpeta reports/.
